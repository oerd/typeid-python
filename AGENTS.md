# AGENTS.md

Python implementation of TypeIDs — type-safe, K-sortable, globally unique identifiers built on UUIDv7. Library code lives in `typeid/`, tests in `tests/`. Lint/format with `ruff` (config in `pyproject.toml`), test with `pytest`.
